EL USUARIO PUEDE ESCOGER LA CANTIDAD DE DATOS QUE DESEE PARA REALIZAR LAS GRÁFICAS
EL USUARIO PUEDE ESCOGER QUE ALGORITMOS DESEA ANALIZAR

-EL PROGRAMA SOPORTA UN ORDENAMIENTO DE 1000 DATOS COMO MÁXIMO.
-EL PROGRAMA TIENE UN TIEMPO LÍMITE DE 3 MINUTOS MIENTRAS SE CARGA LA INFORMACIÓN REQUERIDA PARA LA GRAFICACIÓN CORRESPONDIENTE.
